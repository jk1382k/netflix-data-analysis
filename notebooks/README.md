# 🎬 Netflix Data Analysis

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset using Python.

The analysis focuses on understanding content distribution, genres, ratings, countries, and growth trends to generate meaningful business insights.

---

### 🎯 Objectives

- Understand the Netflix dataset
- Clean and preprocess the data
- Perform Exploratory Data Analysis
- Visualize key trends
- Generate business insights and recommendations
